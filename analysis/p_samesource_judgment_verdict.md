# 同源判断层 + 新特征:walk-forward 证伪"稳健 edge" — 2026-07-23 夜

**纪律**:纯离线,<2026-05-04(不碰 holdout),v16 候选 + 同源判断层。

1. 用 v16 检测器在历史上跑出它自己的盘口候选(密集-only 加速),记录 28 判断特征
   + TP5/SL2/72bar 真实后续收益。**同源**:判断层在 v16 自己的候选上训。

**100 币 / 36 特征,单次 70/30 切分**:top-half PF ~1.09,看似正。

## 结论

**"双均线密集启动"在实时盘口、扣成本、TP5/SL2 结构下,没有稳健可交易 edge。**
盈亏几乎完全由行情 regime 决定(顺风赚、逆风巨亏),而 regime 现有特征预测不了、
同源判断层也捕捉不到。检测层从不是瓶颈;判断层对齐 + 新特征 + 更多数据都不解决——
要预测的"启动"在盘口本身近乎不可测。

- 未穷尽:换出场结构、order flow/资金费率(需重新拉数据)。基于一致性证据翻盘概率低。
