# P0-3 均线 20/60/120 对比实验（val only）

## 复现命令

```bash
# 如需从零重建 8-55 SWAP TP5/SL2 基线数据；本次直接复用 P0-2 已产出的 CSV
PYTHONPATH=. python3 scripts/swap_replication.py

# 生成 20/60/120 数据集并复用现有 LightGBM train/val 流程评估
PYTHONPATH=. python3 scripts/ma206_comparison.py
```

产出：

- `data/ma206/swap_tp5_sl2_ma206.csv`（gitignored）
- `analysis/output/p2b_ma206_comparison.json`

## 实验设置

- 对照组：现行 EMA 8/13/21/34/55 + 144/200 判断层，使用 P0-2 已生成的
  `data/swap_replication/swap_tp5_sl2.csv`；
- 实验组：新增 `src/judgment/candidates_v206.py`，使用 SMA20/60/120 +
  EMA20/60/120 六线；
- 密集规则：SMA/EMA 20/60 四线 `fast_spread <= 0.0028 * 1.6`，
  六线 `full_spread <= 0.0055 * 1.6`，且连续密集至少 5 根；
- volume、pre_range、drawdown、order、slope 等非检测层门槛沿用 expanded 池；
- 标签：TP5/SL2，horizon=72，宇宙为 OKX USDT_SWAP 15m；
- 纪律：只做 train/val；holdout 只由 `load_splits` 计算样本数，未评估。

## 解读

1. **20/60/120 的 AUC 胜出，但收益主标准不胜出。** 它的 val AUC 从 0.560 提升到
   0.626，说明模型更容易区分标签；但 top-decile 毛利只有 +0.093%，扣 0.2%
   往返成本后为 -0.107%，不满足本项目成功标准。
2. **8-55 仍是经济性更强的判断层。** 8-55 的 top-decile 毛利 +0.285%，
   净@0.2% +0.085%，净@maker0.06% +0.224%，显著优于 20/60/120。
3. **单特征基线未超过模型。** 20/60/120 的单特征 `ma_spread_pct` AUC 为 0.524，
   净@0.2% -0.010%；模型确实学到额外结构，但学到的排序没有转化为足够毛利。
4. **AUC 与收益再次分叉。** 这与 v1 教训一致：AUC 不能作为成功标准。20/60/120
   更“可分类”，但 top-decile 不够赚钱。

## 风险与诚实声明

- 本实验没有触碰 holdout；val 已被多轮选型使用，不能把本轮结果当确认性绩效；
- 8-55 基线复用 P0-2 已生成数据集，避免重做合约复制检验；
- 20/60/120 的特征为同构迁移：`close_vs_ema55` 对应 EMA60，`close_vs_ema200`
  对应 EMA120，以保持 `train.py` 不改；
- 资金费率仍是近似成本，P1-7 后需要用真实资金费率复核；
- 20/60/120 若要继续救经济性，已经不是“均线主线替换”本项内的单变量比较，
  应作为新假设单独立项。
