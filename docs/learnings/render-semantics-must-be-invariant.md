# 视觉模型的标签语义必须在渲染层保持不变量

- **问题**：2a 首轮 YOLO 只有 mAP50 0.22，而标注、坐标映射、训练配置反复检查都没错。
- **死胡同**：怀疑标注坐标系不一致、怀疑模型容量、怀疑数据量——都排除了。
- **有效路径**：病根在渲染的 y 轴动态缩放没有下限：横盘窗口的 2-3% 价格极差被放大到
  整图高度，"均线密集"在低波动图里是宽扁带、在高波动图里是细线收拢——同一个数值标签
  对应完全不同的像素模式，模型无法学习。给 y 轴设最小跨度（中位价的 6%）后
  mAP50 0.22 → 0.68，再修框尺寸 → 0.835。
- **通用规则**：把数值规则转成视觉任务时，先验证"规则语义 → 像素模式"的映射是不变量：
  同一规则命中的样本，在任何渲染参数下必须长得像同一类东西。训练指标异常低时，
  先人眼看 20 张不同波动率区间的训练图，再怀疑模型。

  根因分析节。
