# 放宽模型阈值只能增加数量，不能创造交易优势

- **问题**：信号数量偏少时，很容易把 LightGBM 通过率视为收益不足的主要原因，并假设从 q90 放宽到 q80 会改善系统。
- **死胡同**：只比较信号数量或早期几笔盈利，会把同一市场窗口里的样本波动误当成阈值优势；继续在短窗口选择阈值还会产生验证集过拟合。
- **有效路径**：在同一候选、同一时间窗口、同一成本和退出规则下，把 q90-range 与 q80-only 分组裁决。q80 可执行信号增加约 69.8%，但两组 PF 都低于 1，证明瓶颈不只是过滤强度。
- **通用规则**：阈值实验必须同时报告候选漏斗和新增分组的独立扣费经济性；新增组不能覆盖成本时立即拒绝，不与其他参数一起调整。
- **牵连**：`analysis/ma206_q80_shadow_24h_report.md`、`analysis/output/ma206_q80_shadow_24h.json`；ACTIVE、q90、成本和 TP/SL 保持冻结。
