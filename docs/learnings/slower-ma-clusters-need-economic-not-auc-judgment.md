# 慢均线更可分类不等于更赚钱

- **问题**：SMA/EMA20/60/120 的 val AUC 0.6255，高于旧候选族，但 top-decile 在 0.2% 成本下仍为 -0.107%/笔，maker 组合 PF 只有 1.07。
- **死胡同**：把更高 AUC 当成均线替换成功，或者反过来把一次沿用旧 TP5/SL2、h72 的弱收益结果解释成 20/60/120 天生无 alpha。
- **有效路径**：同时查看候选密集持续时间、ATR、top-bucket 毛利、固定成本占比和组合 PF；把架构统一与盈利验收拆成两个结论，新配置只通过新前向账本终审。
- **通用规则**：慢周期信号往往更稳定、更容易分类，但波动和单位时间毛利可能更低；判断它时必须先看扣成本收益和组合 PF，AUC 只作排序诊断。
