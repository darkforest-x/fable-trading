# 合规关闭 HSV 不等于检测指标会提升

- **问题**：旧 E2.1 误留 `hsv_s/v=0.05`，需要按铁律全 0 重训，并判断颜色语义修复能否
  把 mAP50 从约 0.85 推到 0.90。
- **死胡同**：把“修正违规增强”直接等同于“指标一定提升”。增强合规是实验有效性的必要
  条件，不是性能收益保证。
- **有效路径**：保持数据、模型大小、imgsz、batch、epoch 和几何增强不变，只关闭 HSV，
  再独立复验 best.pt 与固定 conf 一致率。mAP50 仅从 0.8503 到 0.8505，一致率仍 0.5127。
- **通用规则**：合规修复先恢复证据可信度；若单变量结果近似不变，应停止重复训练，把
  资源转向标签边界、任务定义或真正控制经济价值的下游目标。
- **牵连**：`src/detection/train.py`、E2.1b args/results/best.pt、
  `analysis/p2a_e21b_hsv0_report.md`；不涉及 judgment holdout 或 ACTIVE。
