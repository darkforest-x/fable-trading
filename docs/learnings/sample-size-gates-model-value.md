# 样本量决定"模型是否比单特征强"——池子小就别怪模型

- **问题**：同一套 v2 特征和 LightGBM，strict 池（2 898 候选）模型 AUC 0.543 反而
  输给单特征 logreg 基线 0.556；expanded 池（10 255 候选）模型 0.565（p=0.001）
  而基线塌到 0.462。
- **死胡同**：如果只跑 strict 池，结论会是"模型没价值，用 ma_spread 单特征就够了"——
  错误归因给方法，实际是样本量不够 LightGBM 学出多维结构。
- **有效路径**：双池对照实验。扩池后基线失效（宽阈值候选质量参差，单特征分不开）
  而模型保持——说明模型学到的增量结构真实存在，只是需要足够样本才显现。
- **通用规则**：评估"模型 vs 简单基线"必须在目标样本规模上做；小样本上模型输给基线
  不构成"模型无用"的证据。反过来：基线在新分布上塌掉而模型稳住，本身就是模型价值的证据。

  HANDOFF"停止做的事"。
