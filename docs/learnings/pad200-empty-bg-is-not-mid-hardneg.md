# pad200 空标背景 ≠ 中段硬负样本

- **问题**：H-DET-2 要压「等后文再框」，但 v13 pad200 已拷了大量空标背景图，容易误以为硬负已经覆盖。
- **死胡同**：把「无框背景」当成硬负——背景没有「有后文的中段密集簇」这个正形状，模型学不到「看见中段簇+后文 → 不要当 tip」。
- **有效路径**：先从现有金标筛 `right∈[0.30,0.90)` 且 `bars_after≥8` 的清单（本仓 v11 train 约 2892 条），协议写明「v13 后再单变量加空标/负类」；开训前必须看预览确认是中段簇而非随机空窗。
- **通用规则**：硬负要保留**易混淆正形状**再标负；空背景只降假阳底噪，不治事后几何。
- **牵连**：`scripts/build_hardneg_mid_cluster_inventory.py`；`analysis/output/hardneg_mid_cluster/`；议程 H-DET-2；勿与 pad200 空标混谈。
