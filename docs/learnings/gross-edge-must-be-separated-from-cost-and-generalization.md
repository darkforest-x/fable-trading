# 毛优势、成本与泛化必须分开诊断

- **问题**：模型 AUC 显著、低成本 maker PF 略高于 1，但保守成本回测明显亏损，容易被
  误判为“模型完全没信号”或“只要降低手续费就能交易”。
- **死胡同**：只看 AUC、top-decile 标签胜率或单个 maker PF，都会把分类能力、毛收益、
  成本侵蚀和行情阶段退化混成一个数字；按最差币种删名单也无法解释广泛退化。
- **有效路径**：先固定同一批交易，做 0/0.06%/0.2%/0.3% 成本桥接，再对比训练期与
  时间外验证期的每笔收益/PF，最后检查分数与实际毛收益相关、分位收益和亏损集中度。
- **通用规则**：收益差时第一步先算“每笔毛优势是否大于可实现成本”，第二步再看时间外
  泛化；AUC 只回答排序标签，不回答能否赚钱。
- **牵连**：`analysis/ma206_profitability_diagnosis.md`、MA206 TP5/SL2、固定 q90 阈值、
  maker/taker 成本假设、前向 100 笔终审纪律。
