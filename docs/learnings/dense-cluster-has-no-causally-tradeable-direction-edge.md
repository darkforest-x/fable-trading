# 双均线密集启动:动作真、方向不可交易 —— 三个结构性杠杆系统性穷尽

**日期**:2026-07-24
**类型**:反直觉结论 / 研究定论(判断层重构实验室 IT-00~IT-13)

## 一句话

v16 能因果检测到真实的双均线密集(both directions),启动后的动作也真实存在
(oracle 完美方向 PF 2.68,+41.9bps),**但在决策时刻无法把这个动作转成可交易的边**
——选点、方向、regime、入场时机四类杠杆全部系统性试完,每条都落在 ~1.0 净值或在最像
实盘的最近一期塌。

## 怎么确认的(每类杠杆都试穿)

| 杠杆 | 迭代 | 结果 |
|---|---|---|
| 选点(判断层 top-decile) | IT-04/06 | 能在期内提纯(period1 到 2.1),但救不了 regime 衰减 |
| 逐笔方向预测 | IT-02 | 28/130/几何特征 AUC 0.47-0.51 = 硬币 |
| 方向规则(全均线上下) | IT-03 | 空头 1.07 / 多头 0.83,thin |
| regime/选边 | IT-05/07/08/10/11 | BTC趋势/波动率门/滚动重训/per-bar/策略动量 **5 角度全墙,最近期塌** |
| 入场 续势(方向无关突破双挂单) | IT-12 | 三期全亏 0.92/0.84/0.83 → 密集启动是**均值回归非续势** |
| 入场 fade(反向挂限价) | IT-13 | ~1.0,fade-long 救 period3 / fade-short 救 period2,**互补翻转依旧** |

## 三个反直觉点(都被数据逼出来)

1. **多空是互补的,但不可利用**(IT-09):空头在前两期强、多头在最近期强 —— 完全互补。
   看起来"按 regime 切边"就能三期都赚,但 regime **翻转点在决策当时不可知**(5 个 causal
   信号全部在翻转段追反)。互补是"多周级慢 regime",per-bar 门把好单切碎。
2. **续势和 fade 两个方向都不赚**(IT-12+13)→ **没有任何 directionless 方式能提取这个动作**。
   加上逐 bar 方向=硬币,方向无关这扇门也焊死了。动作真实(oracle 2.68)但**对称不可预测**。
3. **"间歇性的边"(IT-08)其实是"该换边没换"**(IT-09 揭示),但揭示了也用不上(第 1 点)。

## 为什么这是定论而非"再调一个参数"

- 三类结构性杠杆(选点/方向-regime/入场)+ 两个独立 holdout 否决(#6 全做多 0.78、
  #7 短趋势出场 ~1.0)一致指向同一堵墙,不是单一配置的偶然。
- 每次"救活"都只在某一期或某一边成立,跨三期从未同时 ≥1.3 —— 这正是 curve-fit 的反面证据:
  真边应该跨期稳,这个边换个期就翻脸。

## 教训(给下一个我)

- **oracle/AUC 高 ≠ 可交易**:oracle 2.68 只证明"动作存在",不证明"决策时刻抓得住方向"。
  本项目成功线永远是**因果 walk-forward 三期同时 ≥1.3**,不是 oracle,不是单期漂亮。
- **互补 ≠ 可组合**:两边在不同期各自赚,不等于合起来赚 —— 组合需要"当时就知道该做哪边",
  而那正是不可知的。别被"选对边就三期都好"的事后表骗了。
- **入场时机能试但别指望翻盘**:entry≠eye(Jaccard 0.04)是真问题,但续势/fade 都试过,
  改入场救不回一个本质对称不可预测的动作。
- **穷尽后要敢下定论**:~13 轮后继续调参 = 低价值消耗;诚实定论 + 保留真实价值(检测/告警)
  比强行拟合一个不存在的方向边更负责。

## 仍然真实的价值(没被否定的)

1. v16 检测**能因果识别真密集**(肉眼复核 175 张,框在真收敛上)—— 可做**告警/特征**。
2. **vol 扩张是真的**(oracle 证明)—— 只是方向不可捕获,不能直接下单。
